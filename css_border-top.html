border-top

Описание: Задает границу сверху.

Значения:

border-top: 2px solid #ffffff;

Создание границы слева для элемента. Порядок значений: толщина, стиль и цвет. Возможные единицы: px, em, rem, vh, vw, а так же ключевые слова: thin, medium, thick. Стиль границы: solid, dashed, dotted, double, groove, ridge. Для определения цвета можно использовать ключевые слова названия цвета, шестнадцатеричные значения: #ffffff, RGB: rgb(0, 0, 0), RGBa: rgba(0, 0, 0, 0.9) и currentColor.

border-top: 2px solid;

Создание границы сверху для элемента без указания третьего параметра - цвета. Если параметр цвета не указан, то цвет границы будет таким же как цвет текста элемента.
