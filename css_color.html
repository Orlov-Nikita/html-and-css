color

Описание: Задает цвет текста.

Значения:

color: #ffffff;

Шестнадцатеричное значение цвета.

color: white;

Ключевое имя цвета текста.

color: transparent;

Прозрачный текст.

color: rgb(0, 0, 0);

Цвет текста в RGB формате, где R - красный канал, G - зеленый канал и B - синий канал.

color: rgba(0, 0, 0, 0.5);

Цвет текста в RGBa формате, где R - красный канал, G - зеленый канал и B - синий канал и последнее значение непрозрачность от 0 до 1.
